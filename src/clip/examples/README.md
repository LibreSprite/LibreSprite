# Clip Library Examples

*Last updated: 2025-12-30*

This directory contains example programs demonstrating how to use the clip clipboard library.

## Examples

| File | Description |
|------|-------------|
| **helloworld.cpp** | Basic example that copies "Hello World" to the clipboard and reads it back |
| **copy.cpp** | Demonstrates copying text to the clipboard |
| **paste.cpp** | Demonstrates reading/pasting text from the clipboard |
| **put_image.cpp** | Shows how to copy image data to the clipboard |
| **show_image.cpp** | Demonstrates retrieving and displaying image data from the clipboard |
| **int_format.cpp** | Example of using custom integer data formats with the clipboard |

## Build Support

- **CMakeLists.txt** - CMake configuration for building the examples
- **random.h** - Helper header used by the examples

## Purpose

These examples demonstrate the clip library's API for clipboard operations including text and image handling. They are included as part of the forked/vendored third-party code and serve as reference implementations.
