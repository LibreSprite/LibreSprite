# LibreSprite Documentation

[![License](https://img.shields.io/badge/license-GPL--2.0-blue.svg)](../LICENSE)

This directory contains technical documentation, file format specifications, and third-party license information for LibreSprite.

## Directory Structure

### files/

File format specifications for image formats supported by LibreSprite:

| File | Description |
|------|-------------|
| `ase.txt` | ASE/Aseprite sprite file format specification |
| `fli.txt` | FLI/FLC animation file format specification |
| `msk.txt` | MSK mask file format specification |
| `pic.txt` | PIC image file format specification |
| `picpro.txt` | PIC Pro image file format specification |

These documents describe binary layouts, chunk types, and encoding details for each format.

### licenses/

Third-party library licenses bundled with LibreSprite:

| File | Library |
|------|---------|
| `giflib-LICENSE.txt` | GIFLIB (GIF support) |
| `gtest-LICENSE.txt` | Google Test (testing framework) |
| `libjpeg-LICENSE.txt` | libjpeg (JPEG support) |
| `libpng-LICENSE.txt` | libpng (PNG support) |
| `MIT.txt` | MIT License template |
| `xfree86-LICENSE.txt` | XFree86 (X11 components) |

### plans/

Design documents and implementation plans for upcoming features. Files are prefixed with dates for chronological tracking.
